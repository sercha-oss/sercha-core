# Project Alpha

This is a sample project for testing the LocalFS connector.

## Features

- Sample Go code for search indexing
- Configuration examples
- Documentation for testing

## Getting Started

1. Install dependencies
2. Run the application
3. Search for documents

## Architecture

Project Alpha uses a modular architecture with the following components:

- **API Layer**: Handles HTTP requests
- **Service Layer**: Business logic
- **Data Layer**: Database interactions
